# Project Summary

## Problem

Blind and motor-impaired users face significant barriers when using traditional music players. Most media applications require visual navigation through menus, buttons, and touch interfaces that are inaccessible without screen-based interaction.

## Target Users

- **Blind users** who need screen-free audio interaction
- **Motor-impaired users** who cannot reliably use touchscreens or keyboards
- **Users in hands-free situations** (driving, cooking, exercising)
- **Users who prefer natural voice interaction** over complex interfaces

## Why Screen-Free Interaction Matters

1. **Accessibility**: 2.2 billion people globally have visual impairment (WHO, 2023)
2. **Inclusivity**: Technology should not exclude users based on physical ability
3. **Natural Interaction**: Voice is the most natural human communication interface
4. **Cognitive Load**: Eliminating visual interfaces reduces cognitive overhead
5. **Safety**: Hands-free control in situations where screens are dangerous

## What Echo-Sync Does

Echo-Sync is an inclusive, screen-free Natural User Interface (NUI) media player that provides:

1. **Voice Control** — Direct commands for media playback (play, pause, next, volume)
2. **AI Assistant** — Understands user context and maps it to appropriate music
3. **Guided Assistance** — Helps new users, handles errors, and rejects off-topic requests politely
4. **Smart Ducking** — Automatically reduces music volume during speech interaction
5. **Earcon Feedback** — Audio cues that replace visual indicators for blind accessibility

Echo-Sync combines these features into a unified, screen-free interaction flow where the user can control playback, express a situation or mood, and receive guided help — all through voice alone.
